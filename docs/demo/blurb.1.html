TL;DR: What's this?

Use spaCy + SpanMarkerNER to construct a lemma graph. This is a prelude to inferring the nodes, edges, properties, and probabilities for building a knowledge graph from raw unstructured text source.

  1. use spaCy to parse a document, with SpanMarkerNER LLM assist
  2. build a lemma graph in NetworkX from the parse results
  3. use OpenNRE to infer relations among entities (optional)
  4. run a modified textrank algorithm plus graph analytics
  5. approximate a pareto archive (hypervolume) to re-rank extracted entities
  6. visualize the lemma graph interactively in PyVis
  7. apply topological transforms to enhance embeddings (in progress)
  8. run graph representation learning on the graph of relations (in progress)

Implementation of an LLM-augmented textgraph algorithm for constructing a knowledge graph from raw, unstructured text source.

The textgraph library is based on work developed by Derwen in 2023 Q2 for customer sample apps and also in our Cysoni product. This demo integrates code from:

  • SpanMarkerNER
  • OpenNRE
  • PyTextRank
  • Create a spaCy Visualizer with Streamlit
